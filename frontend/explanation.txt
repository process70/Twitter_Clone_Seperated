first when we go to login page, the isAuthenticated is set to false and the local storage persist value is also set to false thanks to useEffect empty array inside usePersist hook, then after a successful login, the local storage is being changed that will trigger another useEffect inside usePersist which accepts updatePersist as condition lead to update isAuthenticated value, and finally changing isAuthenticated value will trigger useEffect inside App.jsx file this time which logs its value on the console and that will lead us to navigate to profile page since the isAuthenticated is true
